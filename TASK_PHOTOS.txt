ЗАДАЧА: Проставить фото игрокам клуба
======================================

ЦЕЛЬ
----
В системе у игроков команды Легирус 2010 г.р. уже стоят фотографии.
У остальных команд клуба (2009, 2011, 2012, 2013) фото нет — UI показывает
плейсхолдер с инициалами в круге. Надо это исправить.


ИСХОДНЫЙ ЭТАЛОН (за основу)
---------------------------
Открой файл:
    backend/data/players.json

В нём первые 15 строк — игроки команды legirus-2010.
У каждого есть поле "photo" со значением вида:
    "photo": "p17-turapin.png"

ВАЖНО:
* Это ИМЯ ФАЙЛА (с расширением), а НЕ путь.
  НЕ пиши "/assets/players/p17-turapin.png" — только "p17-turapin.png"
* Расширение строго .png
* Имя файла строго совпадает с полем "id" игрока

ПОШАГОВАЯ ИНСТРУКЦИЯ
--------------------

1) Сделай свежий pull, чтобы работать с актуальным состоянием:

      cd "C:\Users\dmuzharov\Documents\Claude\Projects\Экран Легирус"
      git pull origin main


2) Открой backend/data/players.json в любом редакторе с подсветкой
   синтаксиса (VS Code, Notepad++).


3) Пройдись по всем игрокам команд legirus-2009, legirus-2011,
   legirus-2012, legirus-2013 (фильтр по полю "teamId").

   Для каждого:

   а) Возьми его id (например "p11-ozdemir-2011").

   б) Подготовь фотографию:
       - размер: квадрат, например 600×600 или 800×800 px
       - формат: PNG (без потерь, как у эталонных файлов)
       - имя файла: точно по id игрока + расширение .png
         "p11-ozdemir-2011" → "p11-ozdemir-2011.png"

   в) Положи файл в:
        frontend/public/assets/players/p11-ozdemir-2011.png

   г) В players.json замени:
        "photo": null
      на
        "photo": "p11-ozdemir-2011.png"

   д) Если фото на этого игрока нет — оставь "photo": null
      (UI покажет плейсхолдер с инициалами, ничего не сломается).


4) Сохрани файл. Проверь что JSON валиден — VS Code подсветит ошибки
   красным. Если что — Ctrl+Z и попробуй ещё раз аккуратнее.


5) Закоммить и запушь:

      git add backend/data/players.json frontend/public/assets/players/
      git commit -m "feat(players): photos for all club teams"
      git push origin main


6) Через 1-2 минуты Vercel пересоберёт фронт. Зайди под тренером
   на /club → пройди по топам → у всех должны быть фотки вместо инициалов.


ВАЖНЫЕ ПРАВИЛА (НЕ НАРУШАТЬ!)
-----------------------------

* НЕ МЕНЯЙ поле "id" — это связь с матчами. Сломаешь id → сломаешь
  всю статистику игрока.

* НЕ МЕНЯЙ поле "teamId" — это привязка к команде.

* В поле "photo" — ТОЛЬКО ИМЯ ФАЙЛА с расширением:
   ✅  "photo": "p11-ozdemir-2011.png"
   ❌  "photo": "/assets/players/p11-ozdemir-2011.png"
   ❌  "photo": "p11-ozdemir-2011"

* Расширение строго .png — не .jpg, не .jpeg, не .webp. Эталонные
  файлы все PNG, и фронт ожидает именно их.

* Имя файла = id игрока + ".png", без вариаций.
   ✅  p11-ozdemir-2011.png
   ❌  p11-ozdemir.png       (нет суффикса -2011)
   ❌  Ozdemir.png            (не по id)
   ❌  p11-ozdemir-2011.jpg   (не PNG)

* НЕ ДОБАВЛЯЙ фото игроков ДРУГИХ клубов (Пороховчанин, Самсон и т.д.)

* Если что-то пошло не так — откати:
        git checkout backend/data/players.json
   и начни сначала.

СОВЕТ ПО РЕСАЙЗУ
----------------
* Онлайн: https://squoosh.app — закидываешь, выставляешь PNG 600×600,
  скачиваешь.
* Виндовс: правый клик по файлу → Открыть с помощью → Paint →
  Resize → Pixels → 600×600 → Save As PNG.
* Если фото вертикальное (портретное) — вырезай квадрат из центра,
  сохраняя голову/верх плеч. UI круглит фото в круг, поэтому
  важно чтобы лицо было по центру.
